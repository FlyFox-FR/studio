# **App Name**: RememberWhen

## Core Features:

- Contact Creation: Add new contacts with names and associated data.
- Date Input: Store contacts' important dates like birthdays.
- Custom Notifications: Set custom reminders for each contact to trigger notifications ahead of important dates, with intervals configured by the user.
- Reminder Scheduling: Schedule and manage reminders based on user-defined settings, such as '1 week before', '3 days before', or '1 day before' birthdays.

## Style Guidelines:

- Primary color: Soft blue (#90CAF9), providing a calming and friendly feel, inspired by reliability and memories.
- Background color: Very light blue (#E3F2FD), similar in hue but greatly desaturated to create a gentle contrast with the primary color.
- Accent color: Light green (#A5D6A7), an analogous color providing contrast to highlight important interactions, indicating dates.
- Body and headline font: 'PT Sans' for a friendly, readable, and modern appearance.
- Use clean, minimalist icons for navigation and to represent actions, like adding or editing contacts, and setting reminders.
- A clean, intuitive layout to make it easy to add contacts and configure reminders.
- Subtle animations for user interactions to provide visual feedback and make the experience engaging.
